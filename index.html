APP DEL CLIMA

Diseñé y programé una aplicación meteorológica para poner en práctica mis conocimientos en lenguajes front-end, especialmente React.

Para este trabajo, diseñé 12 piezas para usar como imágenes de fondo que describen gráficamente las condiciones climáticas y el horario. Luego desarrollé la aplicación para identificar el clima y la zona horaria de la ubicación solicitada y entregar la información usando su imagen e icono correspondiente.
